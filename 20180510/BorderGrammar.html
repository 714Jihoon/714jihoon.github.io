경계선 정의 순서
(시계 방향) top, right, bottom, left

경계선 스타일 지정
(style) solid, dashed, dotted, hidden, none, …
e.g. border-style:inset; border-bottom-style:solid;

경계선 색상 지정
(color) red, blue, green, …
e.g. border-color:lightgray; border-bottom-color:red;

경계선 너비(=두께) 지정
(width) 10px, thick, medium, thin, …
e.g. border-width:10px; border-left-width:20px;

경계선 모서리 둥근 정도(반지름) 지정
(radius) 5px, 10px, …
e.g. border-radius:5px; border-top-left-radius:5px;

border-bottom-left-radius:10px;
border-bottom-right-radius:20px;
(border-bottom-radius는 안 되는 걸로 보아 위/아래 왼쪽, 오른쪽을 한 번에 쓸 수는 없는 듯)


스타일 축약형 사용 (style 지정은 필수)
width style color
e.g. border: 10px solid gray ;
